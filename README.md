# c3-UNLOCKED

Flutter Android app for browsing media.ccc.de, streaming talks, downloading MP4 files through Android DownloadManager, and watching saved talks later from the app.

## Current Features

- Browse recent talks and conference/event collections from the public media.ccc.de API.
- Search events and open talk detail screens with metadata, tags, poster art, and links.
- Pick preferred MP4 recordings and send downloads to Android public Downloads under `Download/c3-UNLOCKED/...`.
- Track queued/running/completed download state in-app.
- Watch downloaded files or remote recordings with resume-position persistence.
- Futuristic dark C3-style UI with responsive cards and compact Android viewport coverage.

## How This Came to Be

I built this app from scratch because I wanted a mobile-friendly way to enjoy CCC content. After finishing my first version, I discovered ChaosFlix — an older app with a similar idea. ChaosFlix[github.com/NiciDieNase/chaosflix] served its purpose well, but it's showing its age. Rather than letting my work sit unused, I decided to publish this as a modern alternative for today's mobile devices.
